## Installation Instructions

1. **Clone the Repository**  
   Open your terminal and run:
   ```bash
   git clone https://github.com/barabankn22-lab/Ta-ta_cosmetic-shop.github.io.git
   ```

2. **Navigate to the Project Directory**  
   ```bash
   cd Ta-ta_cosmetic-shop.github.io
   ```

3. **Install Dependencies**  
   Ensure you have [Node.js](https://nodejs.org/) installed. Then run:
   ```bash
   npm install
   ```

4. **Run the Project Locally**  
   To start the local server, run:
   ```bash
   npm start
   ```
   The website should be accessible at `http://localhost:3000`.

---

## Netlify Deployment Guide

To deploy the Ta-ta Cosmetic Shop website on Netlify:

1. Create an account on [Netlify](https://www.netlify.com/).
2. Click on **New site from Git**.
3. Authenticate with GitHub and choose the repository you cloned.
4. Set the build command to `npm run build` and the publish directory to `build`.
5. Click on **Deploy Site**.
6. Once deployed, your site will have a live URL that you can share!
